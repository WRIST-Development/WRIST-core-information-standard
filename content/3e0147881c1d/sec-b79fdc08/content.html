1. If Pregnancy Status = Yes, Expected Delivery Date should be populated 2. If CPR Decision exists, CPR Decision Date is required 3. If Safeguarding Flag = Yes, at least one Safeguarding Concern Type required 4. Address must have at least Address Line 1 and Postcode for UK addresses 5. If NHS Number provided, NHS Number Status should be populated ### Temporal Validations 1. Problem End Date must be after Problem Onset Date 2. DoLS End Date must be after DoLS Authorisation Date 3. All dates cannot be future dates unless specifically allowed 4. Review dates must be in the future ### Conditional Requirements 1. If person lacks capacity, Mental Capacity Assessment required 2. If reasonable adjustments needed, details must be provided 3. If safeguarding concerns active, lead professional must be identified
