## Overview This section defines the decision-support logic embedded within WRIST-based systems. These decision tables translate clinical guidelines and best practices into computable rules that provide real-time support for healthcare professionals. Each table specifies conditions based on WRIST data elements and corresponding actions or recommendations. ## DT001: Medication Safety Checking ### Description This decision table implements comprehensive medication safety checks including allergy cross-checking, drug interaction screening, dose range validation, and duplicate therapy detection. It operates whenever medications are prescribed, dispensed, or administered, using the person's allergy profile, current medication list, and clinical parameters. ### Input Elements - MED001 (Medication Code) - MED003 (Dose Amount) - MED005 (Route) - ALG001 (Allergen Code) - ALG003 (Reaction Type) - ALG004 (Reaction Severity) - PRB001 (Problem/Diagnosis Code) - PER004 (Date of Birth) - INV003 (Result Value - for renal/hepatic function) ### Decision Logic | Condition ID | Input Conditions | Output Action | Annotation | |---|---|---|---| | DT001.C01 | New medication substance matches known allergen with severity = severe/fatal | BLOCK prescription, generate critical alert | Prevent anaphylaxis and severe reactions | | DT001.C02 | New medication substance matches known allergen with severity = mild/moderate | WARNING with override option, document reason | Allow clinical judgment for minor reactions | | DT001.C03 | Drug interaction checker returns contraindicated combination | BLOCK prescription, suggest alternatives | Based on BNF interaction tables | | DT001.C04 | Drug interaction checker returns caution required | WARNING with monitoring requirements | Specify required monitoring parameters | | DT001.C05 | Dose exceeds age-appropriate maximum (calculate from DOB) | WARNING with dose range guidance | Reference BNF for Children/Elderly | | DT001.C06 | Renal function (eGFR) < 30 AND medication requires adjustment | Calculate adjusted dose, WARNING | Use Cockcroft-Gault or MDRD formula | | DT001.C07 | Duplicate therapeutic class already prescribed | WARNING for duplicate therapy | Check ATC classification level 3 | | DT001.C08 | Controlled drug prescribed without appropriate registration | BLOCK prescription, registration required | Check prescriber CD authorization | | DT001.C09 | High-risk medication AND no recent monitoring | PROMPT for required blood tests | e.g., warfarin needs INR, lithium needs levels | | DT001.C10 | Pregnancy status = active AND medication category D/X | BLOCK prescription, teratogenic risk | Reference UK Teratology Information Service | ## DT002: Falls Risk Assessment and Prevention ### Description This decision table calculates falls risk scores and triggers appropriate preventive interventions based on multiple risk factors.
