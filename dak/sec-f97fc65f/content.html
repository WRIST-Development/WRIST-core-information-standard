## Overview The WRIST Core Information Standard enables sophisticated clinical decision support through structured data capture and rule-based logic. This section defines key decision tables that transform WRIST data elements into actionable clinical guidance, alerts, and automated workflows. Each decision table specifies input conditions, evaluation logic, and resulting actions. ## DT-001: Frailty Identification and Escalation **Purpose**: Automatically identify frailty indicators and trigger appropriate assessment pathways based on multiple WRIST data elements. **Description**: This decision table monitors key frailty markers including falls history, weight loss, mobility problems, and polypharmacy to identify patients requiring Comprehensive Geriatric Assessment. **Input Elements**: - Age (from Demographics) - Number of falls in past 12 months (from Problem List) - Number of active medications (from Medications) - Recent weight loss >5% (from Examination Findings) - Mobility aid use (from Equipment and Adaptations) -
